Classifier evaluation

Classifier evaluation#

In [1]: from sklearn.ensemble import RandomForestClassifier

In [2]: from sklearn.model_selection import train_test_split

In [3]: from sklearn import datasets

In [4]: from sklearn_evaluation import plot, table

In [5]: import matplotlib.pyplot as plt

sklearn-evluation has two main modules for evaluating classifiers: sklearn_evaluation.plot and sklearn_evaluation.table, let’s see an example of how to use them.

First, let’s load some data and split it in training and test set.

In [6]: data = datasets.make_classification(200, 10, n_informative=5,
   ...:                                     class_sep=0.65)
   ...: 

In [7]: X = data[0]

In [8]: y = data[1]

# shuffle and split training and test sets
In [9]: X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3)

Now, we are going to train the data using one of the scikit-learn classifiers.

In [10]: est = RandomForestClassifier(n_estimators=5)

In [11]: est.fit(X_train, y_train)
Out[11]: RandomForestClassifier(n_estimators=5)

Most of the functions require us to pass the class predictions for the test set (y_pred), the scores assigned (y_score) and the ground truth classes (y_true), let’s define such variables.

In [12]: y_pred = est.predict(X_test)

In [13]: y_score = est.predict_proba(X_test)

In [14]: y_true = y_test

We can start evaluating our model, the following example shows how to plot a confusion matrix.

In [15]: plot.confusion_matrix(y_true, y_pred)
Out[15]: <AxesSubplot: title={'center': 'Confusion matrix'}, xlabel='Predicted label', ylabel='True label'>

Some classifiers (such as sklearn.ensemble.RandomForestClassifier) have feature importances, we can plot then passing the estimator object to the feature_importances function.

In [16]: plot.feature_importances(est, top_n=5)
Out[16]: <AxesSubplot: title={'center': 'Feature importances'}>

A feature importances function is also available in the table module.

In [17]: print(table.feature_importances(est))

Now, let’s see how to generate two of the most common plots for evaluating classifiers: Precision-Recall and ROC.

In [18]: plot.precision_recall(y_true, y_score)
Out[18]: <AxesSubplot: title={'center': 'Precision-Recall'}, xlabel='Recall', ylabel='Precision'>

In [19]: plot.roc(y_true, y_score)
Out[19]: <AxesSubplot: title={'center': 'ROC'}, xlabel='False Positive Rate', ylabel='True Positive Rate'>
